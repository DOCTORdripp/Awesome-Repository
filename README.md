# Decentraland Awesome Repository

Welcome to the Decentraland Awesome repository!

Here you can find content to help you build scenes for Decentraland. Here you'll find links to:

- [Examples](#Examples)
- [Tutorials](#Tutorials)
- [FAQs](#FAQs)

If you can think of an example that is easy to understand and covers valuable topics that aren't covered here, you're encouraged to create a pull request and contribute!

You're encouraged to report issues in the Issues tab of the repositoriy.

See the [Decentraland documentation](docs.decentraland.org) for more detailed information.

## Examples

### Essentials

| Example                                                                            | Description                                                                                                                                                                                              |
| ---------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Sliding Door](https://github.com/decentraland-scenes/Sliding-door)                | Use the Utils library to shift the position of a door gradually. Clicks on the door toggle it from open to closed.                                                                                       |
| [Door](https://github.com/decentraland-scenes/Open-door)                           | Use the Utils library to rotate a door gradually. Clicks on the door toggle it from open to closed.                                                                                                      |
| [Puffer](https://github.com/decentraland-scenes/Puffer)                            | Use the Utils library to scale items and delay actions                                                                                                                                                   |
| [Basic button events](https://github.com/decentraland-scenes/Basic-Interactions)   | A simple example of each way in which players can use button events to interact with the scene. Each shape's color is activated by interacting with it.                                                  |
| [Jukebox](https://github.com/decentraland-scenes/Jukebox)                          | Play different songs by pressing buttons on a jukebox.                                                                                                                                                   |
| [Hypno Wheels](https://github.com/decentraland-scenes/Hypno-wheels)                | A first encounter with Systems, Custom Components and Component Groups.                                                                                                                                  |
| [Shark Animation](https://github.com/decentraland-scenes/Shark-animation)          | Control animations on a 3D model. Toggle them on or off when clicking on the model.                                                                                                                      |
| [Gnark Patrolling](https://github.com/decentraland-scenes/Gnark-patrol)            | A character walks along a fixed path, using lerp over each segment of the path. If you approach it, it will switch states to yelling at you.                                                             |
| [Hummingbirds](https://github.com/decentraland-scenes/)                            | A new bird spawns every time you click a tree. Each bird moves on its own to random positions.                                                                                                           |
| [Smoke](https://github.com/decentraland-scenes/Smoke)                              | Use planes to create a particle system that simulates smoke rising.                                                                                                                                      |
| [Dance Floor](https://github.com/decentraland-scenes/Dance-floor)                  | Combines animations, sound, and tiles on the floor that randomly change color to the beat.                                                                                                               |
| [Laser-Ray Casting](https://github.com/decentraland-scenes/Laser-ray-casting)      | Use ray casting to trace a line in space and check for intersections. Cubes change material when hit by the laser. They also change when being pointed at by the player.                                 |
| [Shark Swimming in Circles](https://github.com/decentraland-scenes/Swimming-shark) | Move a shark along the segments of a curve to swim in circles.                                                                                                                                           |
| [Block Dog](https://github.com/decentraland-scenes/Block-dog)                      | A simple AI character that randomly chooses what action to take: follow you, sit or remain idle. You can also tell it to sit or stand up by clicking it, or tell it to drink water by clicking its bowl. |
